# Starbucks Nutrition & Beverage Analytics Dashboard

## Overview

This Power BI dashboard analyzes Starbucks beverage data to uncover nutritional insights across different beverage categories. The dashboard provides an interactive view of calories, sugar, caffeine, protein, and beverage distribution to support data-driven decision-making.

## Objectives

* Analyze beverage nutritional content.
* Compare calories, sugar, caffeine, and protein across categories.
* Identify high-calorie and high-protein beverages.
* Explore relationships between sugar and calorie content.
* Create an interactive dashboard using Power BI.

## Key Insights

* Coffee-based beverages contain the highest average caffeine levels.
* Certain beverage categories contribute significantly more calories than others.
* High sugar beverages generally tend to have higher calorie values.
* Protein-rich beverages can be identified through category-level analysis.

This dashboard helps users understand the nutritional composition of Starbucks beverages through interactive visualizations and business intelligence techniques.
